# Team Contribution Statement

The AirClass project was developed collaboratively by Team_San through shared coding, testing, and demonstration sessions. Although the main implementation was committed from one development machine for consistency and easier integration, the core design and coding decisions were discussed and implemented as a team. Therefore, the GitHub commit history should not be interpreted as the only measure of contribution; it mainly reflects the repository workflow used during development.
